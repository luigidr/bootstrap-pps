Modale

Un modale (o finestra modale) è un componente che viene utilizzato per mostrare al bisogno informazioni aggiuntive rispetto a quelle presenti nella pagina principale.

Un modale si apre "sopra" la pagina, come se fosse una seconda pagina più piccola a scomparsa. Utilizzi tipici comprendono la comunicazione di dettagli non presenti nella pagina principale o schermate di login/registrazione utenti.

Tipicamente, un modale viene aperto/visualizzato dopo aver fatto click su un link o un bottone.

Puoi vedere le varie opzioni per realizzare un carousel sul sito di Boostrap, all'interno della sezione JavaScript.

Ricerca tramite filtro su un elenco di elementi

Una funzionalità tipica dei siti web è la possibilità di filtrare tra un elenco di elementi (es. alberghi, film, ristoranti) secondo una parola chiave (es. città, genere).

Esempio: ho un elenco di nomi e voglio poter filtrare (quindi ridurre) il numero di risultati visibili, ad esempio selezionando solo le voci dell'elenco che contengono una specifica sequenza di caratteri

Nell'esempio proposto, andando a scrivere nel campo input di testo, è possibile filtrare l'elenco di nomi

function myFunction() {
    // dichiarazione delle variabili
    var input, filter, ul, li, a, i;

    // il valore da usare per filtrare l'elenco √® il testo letto dal campo input
    input = document.getElementById('myInput');

    // decido di filtrare senza fare distinzione tra maiuscole e minuscole
    filter = input.value.toUpperCase();

    // devo prendere tutti gli elementi dell'elenco, quindi prima ottengo l'elenco tramite
    l'id myUl...
    ul = document.getElementById("myUL");
    // ... e poi vado a prendere tutti gli elementi dell'elenco
    li = ul.getElementsByTagName('li');

    // a questo punto ciclo su tutti gli elementi della lista
    for (i = 0; i < li.length; i++) {
        a = li[i].getElementsByTagName("a")[0];

        // se la parola i-esima dell'elenco contiene il valore in input, allora la mostro
        if (a.innerHTML.toUpperCase().indexOf(filter) > -1) {
            li[i].style.display = "";
        // altrimenti la nascondo
        } else {
            li[i].style.display = "none";
        }
    }
}
