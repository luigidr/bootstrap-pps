Template di base

Un template è un file di riferimento da cui partire per realizzare qualcosa, ad esempio un sito web.

Questo è il codice di partenza di una pagina HTML standard: qualsiasi pagina deve avere almeno questo codice.



  <!DOCTYPE html>
  <html lang="en">
    <head>
      <meta charset="utf-8">
    </head>
    <body>
    <body>
  </html>

Elementi

Gli elementi che compongono una pagina sono identificati da un tag di inizio e di fine. Ad esempio:

  • <body></body> definisce il contenuto dell'intera pagina.
  • <div></div> definisce un contenitore all'interno della pagina.
  • <p></p> definisce un paragrafo che contiene del testo
  • <h1></h1> definisce un titolo.

Un elemento può avere dei valori, ad esempio:

  • una classe, <div class="contenitore-1">,
  • un id, o identificativo univoco, <p id="paragrafo-1">.

Un elemento può avere sia classe che id.

Righe

La libreria Bootstrap permette di impostare il layout della pagina, ovvero la disposizione degli elementi, secondo un sistema a colonne, chiamato 'grid'.

Per farlo, come prima cosa, bisogna inserire all'interno del body un elementodiv, ovvero un elemento generico, di tipo container.

All'interno di un container andremo ad inserire una o più righe (row), che potremo quindi suddividere in colonne.

Si possono inserire più container nella pagina, basta fare attenzione a non inserire un container dentro un altro container per evitare problemi di visualizzazione.

NOTA BENE: in questa guida i puntini (...) verranno usati negli esempi per indicare la presenza di codice prima o dopo l'esempio proposto. Non vanno copiati nella vostra pagina! 



  ...
  <body>
    <div class="container">
        <div class="row">
        </div>
        <div class="row">
        </div>
    </div>
    <div class="container">
        <div class="row">
        </div>
    </div>
  <body>
  ...

Colonne

Per aggiungere una colonna è sufficiente aggiungere un elemento generico div con la classe colonna col-md- e la dimensione voluta, ad esempio col-md-1.

In Bootstrap, la pagina si considera di larghezza 12; una colonna può avere una larghezza che va da un minimo di 1 ad un massimo di 12. Una riga non può essere più larga di 12, in caso contrario gli elementi che sforano andranno a capo.



  ...
  <div class="row">
    <div class="col-md-4">
    </div>
    <div class="col-md-8">
    </div>
  </div>

Jumbotron

Un Jumbotron è un elemento di Bootstrap che occupa l'intera larghezza della pagina, con una dimensione di carattere molto grande e che permette di mettere in evidenza contenuti importanti.

Un esempio tipico di utilizzo di un Jumbotron è l'intestazione della pagina (come all'inizio di questa guida).



  ...
  <div class="jumbotron">
    <h1>Un Jumbotron con testo e bottone</h1>
    <p>Una descrizione del Jumbotron</p>
    <p><a class="btn btn-primary btn-lg" href="#" role="button">Premimi!</a></p>
  </div>

Navbar

Un Navbar è un elemento di Bootstrap che tipicamente occupa l'intera larghezza della pagina e che può essere usata come barra di navigazione principale per il sito.

Un Navbar è un componente responsivo, si adatta cioè alla dimensione dello schermo su cui viene visualizzata. In particolare, quando il Navbar è mostrato su uno smartphone (o simili), alcuni suoi elementi vengono racchiusi dentro un bottone denominato hamburger:

 

NOTA BENE: un Navbar può essere personalizzata in tanti modi: può rimanere sempre fissa in cima alla pagina, pu` muoversi con il contenuto della pagina, può avere un'immagine al posto del titolo, ecc. Qui si riporta un esempio semplice ma completo: potete trovare maggiori informazioni e dettagli alla pagina Navbar - Bootstrap.


Elementi di una navbar

Gli elementi più comune di una navbar sono i link di navigazione, che permettono agli utenti di navigare tra le pagine di un sito.

Un link è delimitato dal tag <a></a> e deve avere un campo href=' ' il cui valore deve corrisponder alla url della pagina a cui quel link deve portate.



  ...
  <nav class="navbar navbar-default navbar navbar-fixed-top">
    <div class="container-fluid">
      <!-- Titolo e bottone di toggle sono raggruppati insieme per la visualizzazione su mobile -->
      <div class="navbar-header">
        <button type="button" class="navbar-toggle collapsed" data-toggle="collapse" data-target="#example-navbar-collapse" aria-expanded="false">
          <span class="sr-only">Toggle navigation</span>

Immagini responsive

Le immagini possono essere rese "responsive", cioè adattabili alle diverse dimensioni dello schermo tramite l'utilizzo della classe .img-responsive. La classe equivale ad applicare max-width: 100%; height: auto; display: block; all'immagine.



    ...
    <img src="..." class="img-responsive" alt="Descrizione dell'immagine">
    ...

Centrare elementi

In Bootstrap 3, è possibile centrare immagini responsive utilizzando la classe .center-block.

In maniera simile, si possono centrare paragrafi di testo utilizzando la classe .text-center.



    ...
    <img src="..." class="img-responsive center-block" alt="Descrizione dell'immagine centrata">

    <p class="text-center">Questo testo è centrato!<p>
    ...

Stile

Foglio di stile

Un foglio di stile è un file di testo con estensione .css che contiene tutte le regole grafiche e di stile di un sito.

Le regole di stile da applicare ad un elemento vengono definite secondo il seguente formato:

  1. una parola che identifica l'elemento per cui valgono quelle regole
  2. le regole, delimitate da parentesi graffe
  3. per ogni regola, la parola che la identifica : valore che vogliamo assegnare ;
  4. eventuali commenti, delimitati da /* commento */

Si possono applicare delle regole di stile:

  • all'intera pagina,
  • a tutti gli elementi con lo stesso tag,
  • a tutti gli elementi con la stessa classe,
  • all'elemento con l'id specificato.

NOTA BENE: l'id è un identificatore univoco e per nessun motivo deve essere duplicato all'interno della pagina. Deve essere assegnato ad un solo elemento.



...
/* regole applicate di base a tutta la pagina */
body { /* il formato è regola: valore; */
  color: #4f4f4f;             /* colore del testo */
  background-color: #3c3c3c;  /* colore dello sfondo */
}

/* regole applicate a tutti i div e p */
div, p {
  font-size: 12px;
}

/* regole applicate a tutti gli elementi con la stessa classe */
.classe-base{
  color: #gggggg;
  background-color: #111111;
}

/* regole applciate al solo elemento con l'id specificato */
#identificativo-univoco{
  color: #bbbbbb;
  background-color: #l3l3l3;
}
...

Stile di un contenitore

Lo stile dei contenitori è molto importante per definire il layout e l'aspetto generale del sito. Tra le varie regole, sicuramente vanno considerate:

  • il margin, ovvero la distanza di quell'elemento dagli altri; è possibile distanziare il contenitore rispetto alle 4 direzioni;
  • il border, ovvero il bordo del contenitore, che può avere colore, spessore, stile (continuo, puntini, ecc.), bordo arrotondato;
  • il padding, ovvero la distanza del contenuto dal bordo;

NOTA BENE: quando si indica un valore numeirco è sempre necessario specificare l'unità di misura; la più comune è il pixel, e viene indicata come px; non devono essere presenti spazi tra valore numerico e unità di misura (es. 30px). In altri casi si può usare % come unità di misura, per un valore compreso da 0 e 100, che indicherà un valore percentuale rispetto alla larghezza della pagina.



...
.contenitore-1{
  margin-left: 25%;       /* distanza da sinistra */
  /* pari al 25% della larghezza totale della pagina */

  margin-top: 20px;         /* distanza dall'alto */

  /* se non viene specificato lo spessore,
  il bordo non è visibile */
  border-width: 4px;        /* spessore bordo */
  border-color: #000000;    /* colore bordo */
  border-style: solid;      /* stile del bordo */
                            /* solid corrisponde a linea piena */
  border-radius: 5px;       /* bordo arrotondato */
  padding-top: 35px;        /* distanza del contenuto dal bordo superiore */
  padding-left: 20px;        /* distanza del contenuto dal bordo superiore */
}
...
