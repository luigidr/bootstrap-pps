Stile

Foglio di stile

Un foglio di stile è un file di testo con estensione .css che contiene tutte le regole grafiche e di stile di un sito.

Le regole di stile da applicare ad un elemento vengono definite secondo il seguente formato:

  1. una parola che identifica l'elemento per cui valgono quelle regole
  2. le regole, delimitate da parentesi graffe
  3. per ogni regola, la parola che la identifica : valore che vogliamo assegnare ;
  4. eventuali commenti, delimitati da /* commento */

Si possono applicare delle regole di stile:

  • all'intera pagina,
  • a tutti gli elementi con lo stesso tag,
  • a tutti gli elementi con la stessa classe,
  • all'elemento con l'id specificato.

NOTA BENE: l'id è un identificatore univoco e per nessun motivo deve essere duplicato all'interno della pagina. Deve essere assegnato ad un solo elemento.



...
/* regole applicate di base a tutta la pagina */
body { /* il formato è regola: valore; */
  color: #4f4f4f;             /* colore del testo */
  background-color: #3c3c3c;  /* colore dello sfondo */
}

/* regole applicate a tutti i div e p */
div, p {
  font-size: 12px;
}

/* regole applicate a tutti gli elementi con la stessa classe */
.classe-base{
  color: #gggggg;
  background-color: #111111;
}

/* regole applciate al solo elemento con l'id specificato */
#identificativo-univoco{
  color: #bbbbbb;
  background-color: #l3l3l3;
}
...

Stile di un contenitore

Lo stile dei contenitori è molto importante per definire il layout e l'aspetto generale del sito. Tra le varie regole, sicuramente vanno considerate:

  • il margin, ovvero la distanza di quell'elemento dagli altri; è possibile distanziare il contenitore rispetto alle 4 direzioni;
  • il border, ovvero il bordo del contenitore, che può avere colore, spessore, stile (continuo, puntini, ecc.), bordo arrotondato;
  • il padding, ovvero la distanza del contenuto dal bordo;

NOTA BENE: quando si indica un valore numeirco è sempre necessario specificare l'unità di misura; la più comune è il pixel, e viene indicata come px; non devono essere presenti spazi tra valore numerico e unità di misura (es. 30px). In altri casi si può usare % come unità di misura, per un valore compreso da 0 e 100, che indicherà un valore percentuale rispetto alla larghezza della pagina.



...
.contenitore-1{
  margin-left: 25%;       /* distanza da sinistra */
  /* pari al 25% della larghezza totale della pagina */

  margin-top: 20px;         /* distanza dall'alto */

  /* se non viene specificato lo spessore,
  il bordo non è visibile */
  border-width: 4px;        /* spessore bordo */
  border-color: #000000;    /* colore bordo */
  border-style: solid;      /* stile del bordo */
                            /* solid corrisponde a linea piena */
  border-radius: 5px;       /* bordo arrotondato */
  padding-top: 35px;        /* distanza del contenuto dal bordo superiore */
  padding-left: 20px;        /* distanza del contenuto dal bordo superiore */
}
...

Questo è un div o contenitore che segue le regole css della classe contenitore-1.

Font

È possibile specificare il font utilizzato per il testo di un qualsiasi elemento della pagine e la sua dimensione.



...
h1{
  font-family: Verdana;   /* stile */
  font-size: 75px;        /* dimensione */
}
...

Esempio di titolo h1


I browser web mettono a disposizione alcuni font di default ma per usare font particolari è necessario aggiungere il file corrispondente nella cartella del sito e caricarlo nel file css con il codice mostrato nell'esempio a fianco.

NOTA BENE: i font che si scaricano dal web posso essere in diversi formati, non tutti compatibili con tutti i browser.


...

/* import di un font da google fonts */
@import url('https://fonts.googleapis.com/css?family=Merriweather:400,700');

/* import di un font caricato nella cartella del sito */
@font-face {
  font-family: "Kingthings_Gothique";   /* nome da usare */
  src: url("Kingthings_Gothique.ttf");  /* percorso del file */
}

h1{
  font-family: Kingthings_Gotique;
}
